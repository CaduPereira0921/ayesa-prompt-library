# Socratic Prompting

## O que é

O **Socratic Prompting** usa perguntas estratégicas para guiar o raciocínio — do modelo ou do usuário. Em vez de pedir respostas diretas, o modelo questiona premissas, explora alternativas e aprofunda o entendimento antes de concluir.

## Quando usar

- Levantamento de requisitos onde as necessidades reais não estão explícitas
- Análise de escopo onde as premissas precisam ser questionadas
- Situações onde o cliente ou usuário "sabe o que quer" mas não "o que precisa"
- Validação de decisões antes de implementar

## Como ativar

Adicione ao prompt frases como:
- *"Antes de responder, faça as perguntas necessárias para entender melhor o problema"*
- *"Questione as premissas implícitas nesta solicitação"*
- *"Que perguntas um consultor experiente faria antes de aceitar esse escopo?"*
- *"Identifique o que não foi dito mas que é crítico para a solução"*

## Diferencial cognitivo

O Socratic Prompting é o único que **inverte o fluxo** — em vez de o modelo responder, ele pergunta. Isso é poderoso em contextos onde a pergunta errada gera a resposta certa mas para o problema errado.
