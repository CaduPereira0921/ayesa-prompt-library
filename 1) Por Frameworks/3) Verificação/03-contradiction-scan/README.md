# Contradiction Scan

## O que é

O **Contradiction Scan** instrui o modelo a "reler" seu próprio texto — ou um texto fornecido — em busca de inconsistências internas, afirmações que se contradizem ou premissas incompatíveis.

## Como ativar

- *"Releia o texto abaixo e identifique afirmações que se contradizem entre si."*
- *"Verifique se o escopo descrito na seção X é compatível com o prazo descrito na seção Y."*
- *"Aponte qualquer inconsistência lógica ou de dados neste documento."*
