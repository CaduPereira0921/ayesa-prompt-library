# Exemplo Reference Injection — Especificação Técnica com Normas
> Verificação · Perfil: Analista Funcional / Arquiteto de Soluções

---

Escreva uma especificação técnica para implementação de controles de acesso no SAP baseados no princípio de segregação de funções (SoD).

**Instrução de verificação — Reference Injection:**
Para cada recomendação de controle ou configuração, cite a referência que a embasa:
- Framework ITGC (IT General Controls) relevante
- Norma SOX (Sarbanes-Oxley) aplicável, se houver
- SAP GRC (Governance, Risk and Compliance) best practices
- ISACA / COBIT guidelines

Sinalize com ⚠️ *[recomendação interna — sem norma externa]* quando a prática for boa mas não vinculada a nenhuma norma conhecida.
