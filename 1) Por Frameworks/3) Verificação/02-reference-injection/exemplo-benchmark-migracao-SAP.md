# Exemplo Reference Injection — Análise de Benchmark SAP
> Verificação · Perfil: Arquiteto de Soluções / Consultor Técnico Sênior

---

Elabore uma análise comparativa das opções de migração para SAP S/4HANA (Greenfield, Brownfield e Bluefield).

**Instrução de verificação — Reference Injection:**
Para cada afirmação técnica ou de performance que fizer, indique a referência que a sustenta. Use como fontes:
- Documentação oficial SAP (SAP Help Portal, SAP Notes)
- Guias da SAP Activate Methodology
- Benchmarks publicados pelo Gartner, IDC ou ASUG
- Boas práticas da ASUG (Americas SAP Users' Group)

Se não houver referência conhecida para uma afirmação, sinalize com ⚠️ *[sem referência — validar internamente]* em vez de apresentá-la como fato consolidado.
