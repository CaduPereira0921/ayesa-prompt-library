# Exemplo Counter Example Hunt — Validação de Decisão de Arquitetura
> Verificação · Perfil: Arquiteto de Soluções / Tech Lead

---

A equipe concluiu que a melhor arquitetura para o projeto é uma integração via SAP Integration Suite (middleware centralizado), descartando a integração ponto a ponto.

Antes de fechar essa decisão, aplique o Counter Example Hunt:

1. Busque ativamente **cenários onde o middleware seria a escolha errada** para esse contexto
2. Cite **casos reais ou tipos de projeto** onde integrações ponto a ponto superaram middlewares em performance ou custo
3. Identifique **premissas implícitas na decisão** que, se falsas, invalidariam a recomendação
4. Proponha **o argumento mais forte possível contra** a decisão tomada
5. Após o exercício, avalie: a decisão ainda se sustenta? O que precisaria ser confirmado para mantê-la com segurança?
