## Fluxo recomendado

1. **ROSE** → gere a proposta inicial
2. **Iterative** → refine tom e profundidade por rodadas
3. **Team Prompting** → avalie pelos olhos do cliente, TI e Ayesa
4. **Fact-Checklist** → audite afirmações sem suporte
5. **Contradiction Scan** → verifique inconsistências entre seções
6. **Meta-Cognition** → identifique pontos fracos antes de enviar
