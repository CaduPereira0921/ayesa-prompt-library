## Quando usar

- **RTF:** use quando o formato da documentação é o ponto crítico — gera uma tabela padronizada com ID, cenário, pré-condição, dados de entrada, resultado esperado e status, pronta para incluir em ferramentas como Jira, Confluence ou planilhas de controle
