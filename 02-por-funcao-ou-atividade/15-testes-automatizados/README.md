## Quando usar

- **Iterative (Rodada 3):** o exemplo de refinamento de casos de teste inclui uma rodada específica para conversão dos cenários mais críticos para formato Gherkin (Given/When/Then), pronto para automação com Robot Framework ou similar

## Dica

Combine com o prompt de **Planejamento de Testes (RAIN)** para primeiro definir o escopo e depois automatizar apenas os casos prioritários — evita automatizar testes de baixo valor.
