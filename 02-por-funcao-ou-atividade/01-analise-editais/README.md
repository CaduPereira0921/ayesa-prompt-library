## Quando usar cada um

- **Análise de edital (4Is):** use para fazer a análise inicial de um edital — avaliar se vale concorrer, extrair requisitos eliminatórios e gerar o índice da proposta
- **Verificação (Fact-Checklist):** use após a análise para auditar se os dados extraídos batem com o documento original antes de tomar a decisão de concorrer
