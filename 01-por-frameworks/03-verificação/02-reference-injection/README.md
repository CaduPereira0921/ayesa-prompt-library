# Reference Injection

## O que é

O **Reference Injection** força o modelo a incluir ou citar referências concretas que sustentem as afirmações da resposta — padrões de mercado, normas, benchmarks ou documentação técnica.

## Como ativar

- *"Para cada afirmação relevante, indique qual fonte ou referência a sustenta."*
- *"Cite os padrões SAP, normas ou benchmarks de mercado que embasam sua recomendação."*
- *"Não faça afirmações sem indicar de onde vêm."*
