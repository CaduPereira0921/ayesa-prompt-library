# Counter Example Hunt

## O que é

O **Counter Example Hunt** testa a robustez lógica de uma resposta pedindo que o modelo busque ativamente contraexemplos, exceções ou situações onde a conclusão não se sustenta.

## Como ativar

- *"Você disse que X. Cite casos em que isso não ocorre."*
- *"Busque ativamente exceções à recomendação que acabou de fazer."*
- *"Que cenários invalidariam essa conclusão?"*
- *"Tente refutar sua própria resposta antes de finalizar."*
