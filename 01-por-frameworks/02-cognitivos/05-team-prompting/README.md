# Team Prompting

## O que é

O **Team Prompting** instrui o modelo a assumir **múltiplos papéis simultâneos** — como se diferentes especialistas estivessem analisando o mesmo problema e debatendo entre si antes de chegar a uma conclusão.

## Quando usar

- Decisões complexas que impactam múltiplas áreas (técnica, comercial, operacional)
- Situações onde perspectivas diferentes levam a conclusões diferentes
- Revisão de documentos que precisam passar por múltiplos "filtros" de avaliação
- Quando você quer antecipar objeções antes de apresentar algo ao cliente

## Como estruturar

```
Assuma os seguintes papéis e responda com a perspectiva de cada um:

👤 [Papel 1] — [ponto de vista]
👤 [Papel 2] — [ponto de vista]
👤 [Papel 3] — [ponto de vista]

Após ouvir todos, sintetize uma recomendação final considerando as três perspectivas.
```

## Diferencial cognitivo

O Team Prompting simula **diversidade cognitiva** em uma única interação. É especialmente útil quando você precisa de uma visão 360° sem precisar convocar uma reunião com múltiplos especialistas.
