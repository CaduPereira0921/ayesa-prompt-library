# Exemplo ROSE — Especificação Técnica de Solução
> Grupo 3 · Especificação Técnica · Perfil: Arquiteto de Soluções / Consultor Técnico Sênior

---

🎭 **Role**
Aja como um arquiteto de soluções SAP com experiência em documentação técnica para projetos de implementação, acostumado a escrever para um público misto (desenvolvedores e gestores).

🎯 **Objective**
Produzir uma especificação técnica completa para o desenvolvimento de um relatório customizado no SAP, com base no requisito funcional abaixo.

**Requisito funcional:**
- Relatório de posição de estoque por centro e depósito, com comparativo entre estoque atual e estoque mínimo configurado
- Filtros: centro, depósito, material, faixa de data
- Saída: tela ALV com opção de exportação para Excel
- Módulo: MM (Gestão de Materiais)

🪜 **Steps**
1. Descreva o objetivo técnico do desenvolvimento
2. Liste os objetos SAP envolvidos (tabelas, transações, BAPIs)
3. Defina a lógica de seleção e processamento de dados
4. Especifique o layout da tela ALV (campos, ordenação, totalizadores)
5. Indique os pontos de atenção técnicos e dependências

📦 **Expected Result**
Um documento de especificação técnica que um desenvolvedor ABAP júnior consiga seguir sem precisar pedir esclarecimentos, com todos os objetos SAP nomeados corretamente e a lógica de negócio sem ambiguidades.
